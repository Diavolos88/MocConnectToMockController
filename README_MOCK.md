# Test Mock Service

Тестовая заглушка для демонстрации работы с MockController.

## Описание

Простое Spring Boot приложение с двумя эндпоинтами:
- `/hello` - Hello World эндпоинт с параметризуемой задержкой и ответом
- `/health` - Health Check эндпоинт с параметризуемой задержкой и ответом

## Параметры конфигурации

Параметры хранятся как переменные класса в `MockService`:

### Hello World эндпоинт:
- `delayHelloWorld` - задержка ответа в миллисекундах (по умолчанию: 1000)
- `stringHelloWorldRs` - текст ответа (по умолчанию: "Hello World!")

### Health Check эндпоинт:
- `delayHealthCheck` - задержка ответа в миллисекундах (по умолчанию: 500)
- `stringHealthCheckRs` - текст ответа (по умолчанию: "OK")

## Запуск

```bash
mvn spring-boot:run
```

Приложение запустится на порту 8081.

## Эндпоинты

### GET /hello
Возвращает приветственное сообщение с задержкой.

**Ответ:**
```json
{
  "message": "Hello World!"
}
```

### GET /health
Возвращает статус здоровья сервиса с задержкой.

**Ответ:**
```json
{
  "status": "OK"
}
```
